# E-Commerce System 🛒

A console-based E-Commerce platform built in C++ as an
Object-Oriented Programming semester project.

## Features
- Role-based system: Admin, Seller, and Buyer
- Product management with categories (Electronics, Clothing, Books)
- Order management with dynamic memory allocation
- File I/O for persistent user and product data
- Authentication system with unique username/ID validation

## OOP Concepts Used
- Inheritance (user → admin, seller, buyer)
- Polymorphism and virtual functions
- Abstract classes and pure virtual methods
- Encapsulation with header/source file separation
- Dynamic memory management

## How to Run
- Open in Visual Studio
- Build and run `main.cpp`
- No external libraries needed
